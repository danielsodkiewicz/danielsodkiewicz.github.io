Zendesk + OpenAI Integration



Jan 2025

I have integrated Zendesk with OpenAI to create a working prototype for a support assistant.

Using OpenAI, I built a custom assistant by providing specific instructions and uploading a knowledge base with Q&A information as attached files. This assistant acts as a virtual support agent. I connected it to Zendesk by setting up a trigger that enables communication with the assistant through Internal Notes in tickets. The assistant processes queries and sends responses back as Internal Notes in Zendesk.
